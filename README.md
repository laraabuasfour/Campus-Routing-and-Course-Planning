# Campus-Routing-and-Course-Planning

This project was developed for the Data Structures and Algorithms course.

It implements two main parts:

- Finding the shortest path between buildings using Dijkstra's Algorithm
- Sorting courses based on prerequisites using Topological Sort

## Features
- Load buildings and distances from file
- Compute shortest distance between two buildings
- Print the shortest route and total distance
- Load courses and prerequisites from file
- Sort courses using topological sort
- Print the sorted course plan

## Programming Language
C
